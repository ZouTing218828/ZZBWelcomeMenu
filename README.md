* 点击按钮从下往上弹出界面，并有以下功能
    - 界面显示过程中底部按钮顺时针旋转
    - 新界面中功能按钮按顺序依次弹出
    - 以上个界面作为背景模糊效果显示
    - 点击按钮实现放大消失效果
    - 点击关闭按钮功能按钮逆序弹出
    - 关闭按钮逆时针旋转直到界面消失
